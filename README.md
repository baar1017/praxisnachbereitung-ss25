# praxisnachbereitung-ss25



Zuerst haben wir ein neues Git Repository erstellt und darin einen Ordner "Excel" angelegt. 

In den Ordner haben wir die Datenblätter aus Ilias gelegt. 

Dann haben wir den ersten Commit gemacht und jeder hat das Repository gecloned.
Anschließend haben wir die Tabellen geladen und Formatiert.

Die Tabellen Ausleihen und Geräte haben wir zusammengefügt über die Spalte Gerätenummer. Ausleihen und Mitarbeiterdaten haben wir über Mitarbeiter ID zusammengefügt. 

Alles zusammen wurde in einem neuen Tabellenblatt gespeichert.



Wir haben versucht den Gesamtwert aller Geräte die aktuell nicht im Einsatz sind anzugeben.

Zuerst wollten wir die Aufgaben mit SVerweisen und Formeln lösen. Aber das war sehr kompliziert. 

Dann haben wir Pivot Tabellen verwendet und das war einfacher.



Anschließend haben wir die daten per Python aus der Tabelle gezogen. 

Hier haben wir Arbeitsteilung gemacht. Während ein Teammitglied sich um die Python Datenextraktion gekümmet hat, hat ein anderes Mitglied das Readme geschrieben und commited. 



Danach wurde der Finale Commit von einem anderen PC gemacht. 



Fazit: Excel ist nicht intuitiv bedienbar und es gibt weder guten Community Support noch eine hilfreiche Dokumentation.

Git funktioniert aber als Versionsverwaltung sehr gut. Es ist zwar anfänglich nicht sehr intuitiv, aber wenn man es mal verstanden hat ein gutes Tool.
